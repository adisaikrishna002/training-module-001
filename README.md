# Training Management App (Next.js)

This is a starter Training Management application built with **Next.js** and React.

## Features included
- Training Categories, Training Types, Course Management, Role Mapping, Assessments, File Attachments sample components.
- Global state via React Context + Reducer.
- Theming using CSS variables (styles/theme.css).
- Example pages and search/filter utilities.

## How to run
1. Extract the ZIP and open the folder in VS Code.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```
4. App will be available at http://localhost:3000
